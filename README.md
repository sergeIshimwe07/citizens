# Ecommerce Wap App
Nuxt 3+ Vuetify 3 + Typescript



# Nuxt 3 Starter


## Setup

### Installation

Make sure to install the dependencies

```bash
yarn install
```

### Development

Start the development server on http://localhost:3088

```bash
yarn dev
```

### Production

Build the application for production:

```bash
yarn build
```
